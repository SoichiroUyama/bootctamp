Typography Activity

Welcome to the Typography Activity! In this activity, you will practice applying various typography styles to different elements of a webpage.


Task 1: Apply italics to this paragraph.

Task 2: Transform this paragraph into uppercase.

Task 3: Center-align this paragraph


Fonts and colors

In this activity, we are using the serif font family for this paragraph. Feel free to experiment with different fonts to see how they change the appearance of the text. You can also modify the color properties to choose different colors dfor the text.


Instruction

  1. Select a paragraph and apply a bold font weight to it.
  2. Choose another paragraph and apply a larger font size to it.
  3. Find a sentence that you think should be highlighted, and apply a different color to it.
